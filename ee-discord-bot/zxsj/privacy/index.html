隱私權政策

本隱私權政策說明「伊伊誅仙小幫手」（下稱本服務）如何處理使用者資訊。

1. 收集資訊的範圍

本服務僅在 Discord 平台內部運作，僅使用 Discord API 提供的基本資訊（如用戶名稱、ID、身分組）作為顯示與互動用途。

2. 不會儲存個資

  • 本服務不會將任何使用者資料儲存至外部伺服器。
  • 所有報名資訊僅存在於 Discord 頻道中的訊息 Embed。
  • 不會進行行銷、追蹤、分析或傳送任何資料至第三方平台。

3. 資訊安全

本服務不會主動收集或處理任何敏感資訊，並依 Discord 的安全機制運作。

4. 聯絡方式

若您對本服務有任何疑問或需協助，請透過 Discord 私訊開發者。
