伊伊聊天室介紹

🎉 I. 即時聊天室顯示功能

提供觀眾更有趣、更即時的互動體驗，顯示重要聊天內容

  • 1. 過濾機器人與指令訊息： 聊天室僅顯示重要訊息，不用擔心被洗頻。
  • 2. 連結與圖片顯示： 自動將連結轉換為可點擊的超連結，顯示圖片連結中的圖片。
  • 3. 公告顯示： 支援實況主發佈公告，包含可展開或折疊查看的功能。
  • 4. 標記提醒： 被提及的觀眾會收到醒目效果通知。

✨ II. 即時互動功能

  • 1. 表情反應系統： 可對訊息進行表情反應，即時更新反應數量。
  • 2. 新訊息提示與快捷返回： 當有新訊息時會顯示提示條，提供快速返回訊息底部的按鈕。
  • 3. 簽到公告： 每當累積簽到或連續簽到達到一定天數時，會在聊天室裡公告，增加互動體驗！

🔄 III. 連線功能

  • 1. WebSocket 即時連線： 實現與伺服器的即時互動，包含自動重連、手動連線與中斷。
  • 2. 連線狀態顯示： 即時顯示當前連線狀態及觀眾連線人數。

🎁 IV. 簽到查詢與抽獎功能

  • 1. 簽到查詢： 查詢當日簽到狀態、累積簽到天數與連續簽到天數。
  • 2. 簽到與抽獎互動： 每日簽到後可參與抽獎，有機會獲得虛擬獎品。
  • 3. 抽獎紀錄查詢： 提供最新 30 筆抽獎紀錄查詢，方便用戶了解自己的抽獎結果。

🔍 V. 外部 API 使用說明

  • 1. API 使用： 儲存和管理用戶的互動資料，保障數據安全。
  • 2. 資料安全： 所有資料僅用於內部顯示，確保用戶隱私。

❓ VI. 常見問題解答 (Q&A)

  • Q: 為什麼已簽到但為什麼仍顯示未簽到？
    A: 代表沒有進行授權，請點選伊伊聊天室右上角齒輪【管理存取權】進行授權。 點此查看圖片

📝 變更紀錄

  • 1. 修正網址連結問題：解決訊息中的網址無法點擊的問題。
  • 2. 優化訊息顯示邏輯：避免相同訊息重複顯示，透過檢查機制在顯示前過濾相似內容。
  • 3. 新增簽到提示功能：累積或連續簽到達一定天數後，自動在聊天室顯示互動提示訊息，增強互動性。
  • 4. 增加抽獎頁面授權檢查：啟用抽獎頁面時，檢測用戶是否完成授權，若無授權則彈窗提示，用戶取消授權後將無法開啟抽獎頁面。
  • 5. 修正API：獎品池鍵值缺失問題：補充遺漏的獎品池鍵值，確保抽獎功能正常運作，避免因缺少獎品資料導致抽獎錯誤。
