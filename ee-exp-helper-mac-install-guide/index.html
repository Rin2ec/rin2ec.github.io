伊伊經驗小幫手｜macOS 安裝與啟動指南

作者：凜

若第一次開啟顯示「檔案已毀損」或沒有反應，請依下列流程處理。

快速步驟

1
雙擊 DMG 檔
2
把 伊伊經驗小幫手.app 拖曳到 Applications（應用程式）資料夾
3
開啟 終端機（應用程式 → 工具程式）
4
在終端機輸入下列指令（最後記得加上空格）： 
sudo xattr -r -d com.apple.quarantine
5
在 Finder 把 /Applications/伊伊經驗小幫手.app 拖曳到終端機，路徑會自動補上。
例如： 
sudo xattr -r -d com.apple.quarantine "/Applications/伊伊經驗小幫手.app"
6
按 Enter，並輸入 Mac 管理員密碼。
提示：密碼輸入時不會顯示字元是正常現象。
7
使用下方 替代啟動方式 開啟

替代啟動方式

  1. 在 Finder 右鍵 伊伊經驗小幫手.app → 顯示套件內容。
  2. 進入 Contents/MacOS/
  3. 雙擊 裡面的執行檔（檔名：伊伊經驗小幫手）。
若本機已完成第一次解除隔離（步驟 1～6）， 之後每次要啟動只需使用這個「替代啟動方式」即可，無須再輸入指令。
常見問題

Q1. 指令後面一定要空白嗎？
是。因為要讓 Finder 拖進去的路徑接在後面。

Q2. 為什麼輸入密碼沒有顯示？
macOS 在終端機輸入密碼時不顯示任何字元，直接輸入完成後按 Enter 即可。

Q3. 仍顯示「無法開啟」？
請再確認指令是否成功（無錯誤訊息），若有錯誤訊息歡迎提供給我，或至 系統設定 → 隱私權與安全性 檢查右側是否有「仍要打開」按鈕。

Q4. 為甚麼點擊選擇視窗沒有反應？
因為要開啟螢幕錄製權限，請到 系統設定 → 隱私權與安全性 → 螢幕錄製 找到「終端機」並勾選，然後重新啟動一次即可選取。
